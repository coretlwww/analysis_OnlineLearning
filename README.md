# analysis_OnlineLearning

1. Больше половины пользователей проходят курсы с техническим направлением. 
2. Профессионалы чаще других категорий участников курса забрасывают прохождение курса.
3. Чаще всего курсы забрасывают из-за отсутствия времени. 
4. Больше всего времени курсам уделяют студенты.
5. Пройдет ли курс пользователь, зависит от продолжительности курса - чем дольше курс, тем реже участник курса его завершит.
6.  Аналогично предыдущей гипотезе - чем короче курс, тем быстрее его проходят участники.
7. Оценка за курс не зависит от платформы, на которой размещен курс.
8. Средняя оценка курса - удовлетворительно, то есть 3.
9. Пользователи ставят низкую оценку курсу, когда курс еще не завершен.
10. Количество времени, которое пользователь тратит на курс в неделю, не влияет на завершение курса.
